# Domain Composition

**Prompt:** "Plan a multi-tenant laboratory information system for clinical research sites."

**Setup:** Greenfield planning fixture.

**Expected invariants:** Separates web or API form, SaaS archetype, Research / Lab / LIMS industry, and healthcare or clinical regulatory overlay; loads focused profiles; maps stack candidates without averaging matrices; flags regulatory details for current verification.

**Scoring rubric:** 2 points each for four-axis composition, focused loading, stack mapping, freshness caveat, and no false industry classification. Four-axis composition is a gate invariant.
